# LI.FI Marketing UI Kit

Recreates the li.fi marketing surface (dev/enterprise positioning).

Reconstructed from the live site at li.fi — no codebase was attached. Screenshots + DOM inspection informed layout; tokens come from `../../colors_and_type.css`.

## Components
- `Nav.jsx` — sticky blurred nav with dropdown triggers
- `Hero.jsx` — radial-halo hero w/ gradient headline + dual CTA
- `StatsBand.jsx` — `//by_the_numbers` paired stats
- `ChainsGrid.jsx` — 60+ chains marquee/grid
- `DevCards.jsx` — Widget / SDK / API 3-up
- `EnterpriseGrid.jsx` — 4-up feature grid
- `PartnerQuote.jsx` — quote card w/ role attribution
- `Footer.jsx` — multi-column links

## Screens shown in index.html
Homepage single-page scroll: nav → hero → stats → chains → dev cards → enterprise → partner quote → footer.
